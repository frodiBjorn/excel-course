## Installation

The following steps are required to install the project:

1. Clone the repository using the command: `git clone https://github.com/frodiBjorn/excel-course.git`.
2. Go to the project directory: `excel-course`.
3. Install necessary dependencies: `npm install`.

## Project launch

To start the project run the following command:
npm run start

After launching the project, you can open a browser and go to http://localhost:3000 to see the results.
